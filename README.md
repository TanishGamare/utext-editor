## UtextEditor 

#### This is a project i picked to get Hands on practice on python. 
#### And another reason for picking a texteditor specifically is because i wanna use things that i build to some extent.
 ##### This is build in python with: 

1 . PIL (Python Image Library) - TO give the GUI window an icon.

2 . Tkinter - To create the GUI 

3 . OS - to get the path of the file

#### To run this in your PC you will need the PIL library since it does not come prerequisite with python 
`
python -m pip install pillow`

#### Before installing PIL package  check if you have Tkinter install since the `ImageTK` is the submodule that depends on Tkinter's internal C bindings 

#### If Tkinter (or its dev headers) weren't available when Pillow was built/install, Pillow silently skips building `ImageT`K and you end up with PIL that has `Image` but not `ImageTK`.
